① 導入ルーム（チュートリアル）
目的: 基本操作の理解と目的の確認
攻略方法: 部屋の中にある机（または特定の場所）から**「鍵」**を見つけ出します。 その鍵を使って奥のドアを開けるとクリアです。 (※ここにある時計の時間のズレは、後の部屋のギミックのヒント（伏線）になっています。)
② ループ廊下
目的: 空間の異常（ループ）に気づき、変化を起こして脱出する
攻略方法: ただ真っ直ぐ進むだけでは、同じ廊下を無限にループしてしまいます。
1周目: 廊下にある「箱」などの配置に違和感がないか確認します。
2周目: 同じ場所を通ると「箱」の他に「メモ」が出現しています。このメモ（ヒント）を調べます。
3周目以降: メモのヒントをもとに特定の行動（特定のオブジェクトを調べる等）を起こすことでループから外れ、「扉」が開いて先へ進めるようになります。
③ 回転ルーム（重力操作）
目的: 部屋の構造を変化させて足場を作り出す
攻略方法: そのままでは高すぎてスイッチやドアに届きません。 部屋の中央付近にある**「回転装置」**を起動します。 部屋自体が90度回転し、「壁」だった場所が「床」になるなど重力の方向が変わります。この性質を利用して、手が届かなかったスイッチを押してドアを開きます。
④ 視点トリック部屋
目的: カメラの視点（錯覚）を利用して隠された道を開く
攻略方法: 部屋にはバラバラに配置されたオブジェクト（A、B、C）があります。 特定の**「視点（カメラの角度や立ち位置）」**からこれらのオブジェクトを覗き込みます。 視点をピッタリ合わせ、複数のオブジェクトが「一つの意味のある形（マークなど）」に見重なった瞬間、ギミックのロックが解除されて道が開きます。
⑤ 分身ギミック室
目的: 過去の自分と協力して2つのスイッチを同時に押す
攻略方法: 部屋には「スイッチA」と「スイッチB」があり、同時に押さないと先へ進めません。
部屋にある**「記録装置」**を起動します。
制限時間内に「スイッチA」の前に移動して待機する行動をとり、記録を終了します。
記録を**「再生」**すると、先ほどの自分の行動を行う「分身（半透明のプレイヤー）」が現れます。
分身が「スイッチA」に乗るタイミングに合わせて、自分自身は「スイッチB」に乗ります。これで同時押し成功です。
⑥ 最終統合ルーム
目的: これまでの全ギミックを複合させてクリアする
攻略方法: これまでの知識を総動員する最終パズルです。以下の手順を踏む必要があります。
分身セット: まず「記録装置」を使い、自分の動き（回転装置の起動や特定位置への移動）を記録させます。
回転: 分身（または自分）の操作で部屋を回転させ、地形を変化させます。
視点合わせ: 回転後の地形で、特定の立ち位置から「視点トリック」の形を合わせます。
同時押し: 視点が合った状態で出現する（または起動可能になる）スイッチを、分身とタイミングを合わせて**「同時押し」**します。 これを突破すればEXIT（脱出成功）です！

「Project Loop Room」は、空間そのものを使った3D脱出ゲームです。
**「理解＝攻略」**をコンセプトとし、プレイヤーが空間の法則やギミックに“気づいた瞬間”を楽しむゲームデザインとなっています。

## 🗝️ 各部屋の攻略方法（ネタバレ注意）

### ① 導入ルーム（チュートリアル）
- **目的:** 基本操作の理解と目的の確認
- **攻略方法:** 部屋の中にある机から**「鍵」**を見つけ出し、奥のドアを開けるとクリア。*(※時計の時間のズレは後のギミックの伏線)*

### ② ループ廊下
- **目的:** 空間の異常（ループ）に気づき、変化を起こして脱出する
- **攻略方法:**
  - **1周目:** 廊下にある「箱」などの配置に違和感がないか確認する。
  - **2周目:** 「箱」の他に出現している「メモ（ヒント）」を調べる。
  - **3周目以降:** メモのヒントをもとに特定の行動を起こすことでループから外れ、**「扉」が開く**。

### ③ 回転ルーム（重力操作）
- **目的:** 部屋の構造を変化させて足場を作り出す
- **攻略方法:** 中央付近の**「回転装置」**を起動して部屋を90度回転させる。壁が床になるなど重力の方向が変わる性質を利用し、通常では届かなかったスイッチを押してドアを開く。

### ④ 視点トリック部屋
- **目的:** カメラの視点（錯覚）を利用して隠された道を開く
- **攻略方法:** バラバラに配置されたオブジェクト（A、B、C）を特定の**「視点（カメラの角度・立ち位置）」**から覗き込む。視点が合わさって一つの形に見えた瞬間、ロックが解除されて道が開く。

### ⑤ 分身ギミック室
- **目的:** 過去の自分と協力して2つのスイッチを同時に押す
- **攻略方法:**
  1. **「記録装置」**を起動し、スイッチAの前に移動して待機する行動を記録する。
  2. 記録を**「再生」**し、自分の過去の行動を行う「分身」を出現させる。
  3. 分身がスイッチAに乗るタイミングに合わせて、自分はスイッチBに乗る（同時押し成功）。

### ⑥ 最終統合ルーム
- **目的:** これまでの全ギミックを複合させてクリアする
- **攻略方法:**
  1. **分身セット:** 「記録装置」を使い、自分の動き（回転装置の起動など）を記録させる。
  2. **回転:** 分身（または自分）の操作で部屋を回転させ、地形を変化させる。
  3. **視点合わせ:** 回転後の地形で、特定の立ち位置から「視点トリック」の形を合わせる。
  4. **同時押し:** 視点が合った状態で出現するスイッチを、分身とタイミングを合わせて**「同時押し」**する。これを突破すれば脱出成功。

## 🎮 コンセプト

- **空間そのものを使った3D脱出**: 単なるアイテム探しではなく、部屋の構造や法則を解き明かします。
- **「理解＝攻略」の構造**: テキストで過剰に説明せず、ギミックを通してプレイヤー自身に気づかせます。
- **ストレス排除**: 理不尽な謎を避け、「失敗＝学習」となるように調整されたゲームプレイ。

## 🗺️ 全体フロー（進行構造）

ゲームは以下のエリアを順に進行し、最終的にすべてのギミックが統合された部屋を突破することでクリアとなります。

1. **① 導入ルーム (チュートリアル)**: 基本操作とゲームの目的を理解する部屋。
2. **② ループ廊下**: 状況が変化するループ空間。特定の条件を満たすことでループから抜け出せます。
3. **③ 回転ルーム**: 重力操作と部屋の90度回転を利用して進む部屋。
4. **④ 視点トリック部屋**: 特定の視点（カメラの角度）から見ることで道が開けるギミック。
5. **⑤ 分身ギミック室**: 自身の行動を記録・再生し、過去の自分と協力してスイッチを同時押しする部屋。
6. **⑥ 最終統合ルーム**: これまでの全ギミック（回転・視点・分身）を活用して解く最終パズル。
7. **EXIT**: 脱出成功！

## 💻 開発環境と仕様

- **エンジン**: Godot Engine (v4.6.2-stable_mono_win64)
- **解像度**: 1920 x 1080 (1080p)
- **物理エンジン**: Jolt Physics
- **メイン言語**: C# / GDScript
- **メインシーン**: `res://Scenes/Rooms/Room1.tscn`

## 🧱 コアシステム設計

- **GameManager (Autoload)**: ループ回数、進行状態、シーン遷移の全体管理を行います。
- **SaveManager**: プレイヤーの現在位置や状態、フラグなどをJSON形式で保存・読み込みします。
- **GimmickManager**: 回転、視点判定、分身再生などの各部屋のコアギミックを制御します。

### 💾 セーブデータ構造例
```json
{
  "scene": "Room3",
  "playerPos": [0.0, 1.0, 30.0],
  "loopCount": 2,
  "flags": {
    "door1": true
  }
}
```
